## Description:-

A React Web Application basically a clone of youtube.
In this project, I have used Axios Library for fetching the API of youtube
by giving unique API Key which we can get it from Google Developer Console.

## Tech Stack:-

1. ReactJs
2. JSX
3. API
4. CSS

## Features:-

1. Responsive for all devices.
2. Search Bar for search any videos.
3. Shortcut menu for go to the respective categories.
4. Youtube Autoplay and Fullscreen Mode Feature.
